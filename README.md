## 📌 About the Project

This portfolio was developed to:

* Present professional background and technical competencies
* Showcase selected development projects
* Demonstrate architectural organization and clean code practices
* Provide direct contact channels
* Maintain a scalable and deploy-ready structure

The application follows modern frontend development standards, focusing on modular architecture, performance, and maintainability.
